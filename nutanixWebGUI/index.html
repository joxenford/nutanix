The Nutanix Prism GUI is the Graphical User Interface that provides
Administrative and monitoring capabilities for the Nutanix Virtual Computing Platform.

The following web pages will provide a very basic overview of the Nutanix Prism GUI

This is in no way an actual representation of the Prism GUI and is not tied
to a live system. It is for demonstration purposes only so you can
get an idea of the various features and functions of a real system.

Functionality is still very limited and certain items may not work at all.

When in doubt, click/tap the in the top middle of the screen
to return the the main Dashboard view.

This was created as a sales tool to show potential customers some of the
screens and features of our latest Web Console.
